# Krisenfestigkeit Monitor – Phase 3

Bolt-fähige React/Vite-Anwendung zur Bewertung der Krisenfestigkeit von Unternehmen mit Branchenmodulen, Maßnahmenmanagement, Nachweisbibliothek und interner KRITIS-Auditlogik.

## Start
```bash
npm install
npm run dev
```

## Enthalten
- Grundanalyse über 8 Resilienzdimensionen
- Branchenmodule per JSON
- Maßnahmen- und Evidenzmanagement
- Governance & Struktur
- KRITIS-Readiness, Audit-Checklist und Feststellungen
- Reporting und Exporte

## Hinweise für Bolt
- Dateien entpackt in ein leeres Projekt ziehen
- `.npmrc` ist enthalten
- kein `package-lock.json` mitliefern
